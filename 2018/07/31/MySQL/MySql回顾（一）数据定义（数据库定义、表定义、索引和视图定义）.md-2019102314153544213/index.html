1.2 参数及注意事项

  • [temporary] 创建临时表。
  • [engine=引擎类型]可选引擎类型
  • 创建数据库前要进行选择数据库，否则会报错，选择数据库语法 use 数据库名。

1.3 完整性约束

  • primary key 标识主键
  • foreign key 标识外键
  • not null key 标识不能为空
  • unique 标识该属性是唯一
  • auto_increase 标识该属性值是自动增长
  • default 为该属性设置默认值
